Help for Wanderer

Wanderer was written by Steven Shipway in the 1980's. See The original credits.
This version was written by Ian Holyer in December 2011: see the design. The controls are:


arrow keys   to move (or h,j,k,l or w,a,s,d)
space bar to stand still
menu to choose next level or restart level

To play the game, collect all the stars and then go through the exit. The entities you will meet are:


This is you.
Exit - must collect all the stars first.
Solid wall.
Solid rock.
Passable earth (score 1).
Star. Collect them all before exiting (score 10).
Boulder (falls down, other boulders and arrows slide round it).
Arrow, running left.
Arrow, running right.
Balloon - rises, and is popped by arrows. It does not kill you.
Landmine. Instant annihilation
Deflector. Boulders etc. slide past.
Deflector.
Teleport (score 20).
Time capsule, giving 250 extra moves (score 5).
Monster (eats you up whole. Yum Yum yum..) (kill with a rock or arrow, score 100).
Baby monster (kills you, but doesn't chase you). Capture it in a cage (score 20) to turn it into a star.
Cage - traps baby monsters.
